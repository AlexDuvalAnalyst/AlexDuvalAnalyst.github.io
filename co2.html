Analyse des émissions de Co2

Un dashboard créé sur Power bi, avec un pré traitement des données sur Python

Les données publiques sont issues du site → ourworldindata ←
et correspondent aux différentes sources d'émissions de Co2 par pays et par années.

Un projet Power Bi

→ Power BI ← est un outil de visualisation et d'analyse de données, produit par Microsoft. Ses nombreuses fonctionnalités permettent la création de tableaux de bord ou dashboard intéractifs, avec graphiques, tables, cartes, etc...
Les données issues de la plateforme → ourworldindata ←, sont mises à jour régulièrement, et ces dernières ont été traitées sur Python (Pandas), permettant ainsi l'extraction d'une partie du jeu de données et des variables associées.

La présente analyse a pour but de revenir sur les émissions de Co2 mondiales des 60 dernières années, notamment pour mettre en évidence Les principales sources de Co2, les pays les plus polluants (rapport par nombre d'habitants), ainsi que les dynamiques évolutives pour chaque pays.
