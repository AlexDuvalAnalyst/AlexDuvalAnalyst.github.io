Une petite application pour smartphone, destinée à géo-localiser l'utilisateur pour estimer les horaires de marées (Pleine mer et Basse mer), au plus proche de sa position (sur la Seine).

Une application mobile (pour les pêcheurs principalement !)

Cette application n'est pas vraiment formatée pour être affichée sur un ordinateur, puisqu'elle est destinée à un format mobile, principalement pour un public de pêcheurs. L'objectif est simplement de renseigner l'utilisateur sur les horaires de Pleines et Basses mer, peu importe sa localisation sur la Seine.
Une autre ambition pourrait être donnée à cette application : étendre le champ d'action à tous les principaux cours d'eau de France, et intégrer des données météo.

Un traitement des données sous Python et une construction via shiny mobile

Pour ce projet, les données ont dans un premier temps été exploitées et traitées sous Python (Pandas), puis intégrées dans l'application. Cette dernière a été contruite grâce au package Shiny mobile, un dérivé du package Shiny dans R.
Le principal défi pour la construction de cette appli, consiste en la récupération des données GPS de l'utilisateur, puis l'interpolation des horaires de marées en fonction d'un quadrillage sur la Seine, ainsi que de sa position.
